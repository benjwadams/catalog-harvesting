catalog_harvesting.cli
======================

Usage::

    usage: cli.py [-h] [-s SRC] [-d DEST] [-v] [-f]

    Command line interface for pulling a WAF

    optional arguments:
      -h, --help            show this help message and exit
      -s SRC, --src SRC     Source WAF
      -d DEST, --dest DEST  Destination Folder
      -v, --verbose         Enables verbose logging
      -f, --force-clean     Removes stale contents of the folder
